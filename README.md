# 🧠 Quiz App

An interactive Quiz Application built using HTML, CSS, and JavaScript.  
This app allows users to answer multiple-choice questions and get their score instantly.

---

## 🚀 Features
- Multiple choice questions
- Instant score display
- Responsive design
- Clean and user-friendly UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
